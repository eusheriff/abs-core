# Core Components (Stubs)

Esta pasta contém a estrutura de referência para a implementação do ABS Core. Na versão v0.1, estes componentes são apenas stubs ou definições de interface.

## Componentes

- **event_ingestion**: Entrada e validação de eventos.
- **process_orchestrator**: Gerenciamento de estado (Sagas).
- **state_machine**: Framework agnóstico de máquinas de estado.
- **policy_engine_stub**: Mock do motor de regras.
- **audit_logger**: Interface de log imutável.
